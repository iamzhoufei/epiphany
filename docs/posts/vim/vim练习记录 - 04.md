## 更有效率地处理单个字符

### 操作符们

- `s` → 删除光标当前字符 并进入 `insert` 模式

- `S` → 删除光标所在行的所有字符 并进入 `insert` 模式

- `r` → 选中当前光标所在字符，等候新键入的字符，并使用新建入的字符替换当前字符

- `R` → 以当前光标所在的字符为起点，等待新建入的字符，并使用新建入的字符依次替换当前字符及后续字符

- `x` → 删除当前光标所在字符

- `X` → 删除当前光标所在字符的前一个字符

### undo & redo

- `u` → 撤销上次操作

- `ctrl` + `r` → 重新执行上次操作
